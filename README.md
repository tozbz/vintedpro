# VintedPro — Dashboard de gestion Vinted/Temu

Application web pour gérer ton business de revente Vinted.

## Fonctionnalités
- Dashboard avec KPIs en temps réel
- Gestion des commandes Vinted
- Suivi du stock Temu
- Calculateur de marge automatique
- Top niches 2026
- Générateur d'annonces IA
- Compta avec export CSV

---

## Déploiement sur Vercel (5 minutes)

### Étape 1 — Mettre le projet sur GitHub
1. Va sur [github.com](https://github.com) → **New repository**
2. Nomme le repo `vintedpro`
3. Upload tous les fichiers du projet (glisse-dépose)
4. Clique **Commit changes**

### Étape 2 — Déployer sur Vercel
1. Va sur [vercel.com](https://vercel.com)
2. Connecte ton compte GitHub
3. Clique **Add New Project**
4. Sélectionne ton repo `vintedpro`
5. Clique **Deploy** — c'est tout !

### Étape 3 — Accéder à ton app
Vercel te donne une URL du style :
`https://vintedpro-xxx.vercel.app`

Tu peux y accéder depuis ton téléphone ou PC, partout.

## Données
Les données sont stockées dans le **localStorage** du navigateur.
Elles persistent entre les sessions sur le même appareil/navigateur.
